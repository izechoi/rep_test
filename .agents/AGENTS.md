# 프로젝트 규정 (Project Rules)

사용자의 지시에 따라, 본 프로젝트 개발 시 아래의 행동 가이드라인을 최우선으로 반영합니다.

1. **무조건 즉시 진행 및 자동 승인 (최우선)**
   - 파일 수정, 삭제, 라이브러리 설치, 또는 주요 설계 변경 시 사용자에게 승인을 구하는 질문이나 대기 상태를 가지지 않고 **즉시 자동으로 구현 및 반영**합니다.
   - 마크다운 아티팩트로 계획을 수립하고 피드백을 요구하는 대기 단계를 생략하고 즉시 코드를 변경 및 실행합니다.

2. **한국어 답변 원칙**
   - AI 에이전트의 모든 생각(Thought) 및 답변은 자연스러운 한국어로 작성합니다.
   - 초보 사용자를 배려해 상세한 한글 주석과 복사 가능한 전체 코드를 항상 제공합니다.
